# NumPy Project: Banking Transaction Analysis

## Project Overview

This project analyzes daily banking transaction data using NumPy. The objective is to evaluate transaction activity and generate useful insights through statistical analysis.

## Problem Statement

Banks process thousands of transactions every day. Analyzing transaction data manually is time-consuming and inefficient. This project provides a simple way to analyze daily transaction data and gain useful insights into banking activity.

## Objectives

* Store daily transaction data using a 1D NumPy array.
* Calculate total and average transactions.
* Identify the highest and lowest transaction days.
* Measure transaction variation using transaction range.
* Calculate the median transaction value.
* Determine the number of above-average transaction days.

## Tools Used

* Python
* NumPy
* Jupyter Notebook

## Key Analysis

* Total Transactions
* Average Transactions
* Highest Transaction
* Lowest Transaction
* Transaction Range
* Median Transactions
* Above Average Transaction Days

## Insights

* A total of 11,100 transactions were processed during the week.
* The average number of transactions was around 1,586 per day.
* The minimum transactions were 1,100, while the maximum transactions were 2,200.
* Transaction activity varied by 1,100 transactions between the highest and lowest days.
* A typical day processed around 1,500 transactions based on the median value.
* Three days recorded transaction volumes above the weekly average.

## Business Impact

* Helps monitor banking activity.
* Identifies peak and low transaction days.
* Supports resource and cash-flow planning.
* Helps detect unusual transaction patterns.
* Enables data-driven decision making.
* Reduces manual analysis effort.

## Conclusion

This project demonstrates how NumPy can be used to analyze banking transaction data efficiently. By calculating key metrics such as total transactions, average transactions, median transactions, transaction range, and above-average transaction days, banks can gain valuable insights into transaction patterns and improve operational planning.
